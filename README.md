# AIKI - Autonomous Intelligent Knowledge Interface

**Network-level ADHD accountability system med emergent AI.**

## Mappestruktur

```
aiki/
├── src/           # Kildekode (AIKI Prime, Circles, Mini-AIKIs)
├── bin/           # Kjørbare shell scripts
├── tools/         # Utvikler- og admin-verktøy
├── services/      # Daemons og systemd-filer
├── data/          # Runtime data og session state
├── config/        # Konfigurasjon
├── docs/          # Dokumentasjon
├── tests/         # Tester
└── _ARCHIVE/      # Arkiverte/utdaterte filer
```

## Hurtigstart

```bash
# Start AIKI-HOME proxy
./aiki-home/start_aiki_home.sh

# Sjekk system health
python tools/system_health_dashboard.py
```
